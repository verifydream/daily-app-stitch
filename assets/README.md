# Assets Directory

This directory contains your app's assets like images, icons, and splash screens.

## Required Assets

For a production-ready app, you should replace the placeholder files with actual assets:

- **icon.png** - App icon (1024x1024 recommended)
- **splash.png** - Splash screen image (1284x2778 for iOS, scale accordingly)
- **adaptive-icon.png** - Android adaptive icon (1024x1024)
- **favicon.png** - Web favicon (48x48 minimum)

## Generating Assets

You can use tools like:
- [Expo's asset generator](https://docs.expo.dev/guides/app-icons/)
- Figma or Sketch for design
- Online icon generators

## File Formats

- PNG format is recommended for icons
- Keep original high-resolution versions
- Expo will handle resizing for different platforms
